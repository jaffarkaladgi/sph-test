# How to use qr code for product module.

- Git clone module under "web/modules/custom"
- Enable module
- Run command "composer require chillerlan/php-qrcode"
## What it does by its own
    - It will create content type for purchase online products.
    - It will create custom block for generating the qr code.
    - Qr code generation Custom block will be placed at the right sidebar .
